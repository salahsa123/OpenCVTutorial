This repository contains two simple C++ examples using OpenCV.
They demonstrate how to:

Capture and display live video from a webcam.

Create an empty image and draw a circle on it.
Uses VideoCapture(0) to access the default camera

Displays frames using imshow()
  in circle                                       
Creates an empty black canvas using cv::Mat::zeros()

Draws a circle using cv::circle()

Displays image in a window

Waits for any key press before closing
